# SQL & Database Design (ERD)

## Objective
Design a relational database for a business case and demonstrate SQL querying
skills through clean schema design, ERD modeling, and analytical queries.

## Skills Demonstrated
- ER modeling (entities, relationships, constraints)
- Normalized schema design
- SQL querying (joins, aggregation, filtering, subqueries)

## Tools
SQL (MySQL/PostgreSQL) + diagram tool (Lucidchart/Miro/Mermaid)
